# CRKP Risk Prediction Calculator

Web-based calculator for predicting carbapenem-resistant *Klebsiella pneumoniae* (CRKP) risk using machine learning.

This tool is designed to support clinical and research decision-making by estimating the probability of CRKP infection based on patient and clinical features.

---

## Quick Start

### Option 1: Use Online (Recommended)

Access the live web applications:


- **Deployment:**

### Option 2: Run Locally

Follow these steps to run the application on your local machine:

```bash
# Clone the repository
git clone https://github.com/maroofb88/CRKP_RISK_Calculator.git
cd CRKP_RISK_Calculator

# Install dependencies
pip install -r requirements.txt

# Launch the Streamlit app
streamlit run streamlit_app.py
